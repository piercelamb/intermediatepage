Introducing SnappyData

A new real-time analytics platform that combines probabilistic data structures, approximate query processing and in memory distributed data management to deliver powerful analytic querying and alerting capabilities on Apache Spark at a fraction of the cost of traditional big data analytics platforms. Learn more on our blog

  • In-memory synopsis store

    SnappyData is an in-memory, distributed store that creates synopsis data to deliver faster analytics queries on less hardware.

  • Spark & SQL interfaces

    SnappyData delivers ease of use by exposing the Spark programming model and SQL query execution.

  • Streaming+Historical Analytics

    Use low latency approximate querying techniques in both historical and streaming contexts.

How does it work?

SnappyData creates probabilistic summary structures and stratified samples on ingested data and intelligently uses these to answer SQL queries on the complete data set. The use of Apache Spark as the underlying platform allows data scientists and application developers to take advantage of the Spark programming model and ecosystem. Learn more on our blog
